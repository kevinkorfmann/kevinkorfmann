# Kevin Korfmann

Computational population geneticist working at the intersection of deep learning, coalescent theory, and simulation-based inference.

I build tools and methods for understanding evolutionary processes — from graph neural networks on ancestral recombination graphs to latent diffusion models for coalescent translation.

### Selected Projects

**Deep Learning & Genetics**
- [GNNcoal](https://github.com/kevinkorfmann/GNNcoal) — Graph neural networks for coalescent inference
- [coalescence-transformer](https://github.com/kevinkorfmann/coalescence-transformer) — Transformer models for coalescent data
- [slinkage](https://github.com/kevinkorfmann/slinkage) — Self-supervised population genetics via linkage
- [arglearn](https://github.com/kevinkorfmann/arglearn) — Geometric deep learning for ancestral recombination graphs
- [cxcld](https://github.com/kevinkorfmann/cxcld) — Latent diffusion for coalescence translation

**Simulation & Inference**
- [CanningsSimulator](https://github.com/kevinkorfmann/CanningsSimulator) — Cannings model simulator for large-variance offspring production
- [subSLiMe](https://github.com/kevinkorfmann/subSLiMe) — Non-linear SLiM scaling correction
- [methylSBI](https://github.com/kevinkorfmann/methylSBI) — Simulation-based inference for methylation data
- [sleepy](https://github.com/kevinkorfmann/sleepy) — Selective sweeps under the weak seed bank model

**Tools & Utilities**
- [fastcxt](https://github.com/kevinkorfmann/fastcxt) — Fast coalescent translation toolkit
- [essaim](https://github.com/kevinkorfmann/essaim) — Distributed AI agent system
- [sgp](https://github.com/kevinkorfmann/sgp) — Simple Genomic Processor (SRR to GVCF)
